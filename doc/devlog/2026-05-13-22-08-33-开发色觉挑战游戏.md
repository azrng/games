# 2026-05-13 开发色觉挑战游戏

## 本次目标
- 按 `doc/requirement.md` 中的色觉挑战需求，实现纯移动端 Kuku Kube 风格小游戏并接入游戏平台目录。

## 核心改动
- 新增 `color-challenge/index.html`、`style.css`、`script.js`，实现自动开局、色块生成、轮次递进、正确/错误判定和结算弹层。
- 使用 CSS Grid 渲染色块，不使用 Canvas。
- 按轮次配置 2×2 到 6×6 网格，并让色差随轮次逐步缩小且保留最低可辨识下限。
- 使用 `color-challenge-high-score` 和 `color-challenge-best-rounds` 保存最高分与最长存活轮数。
- 在 `data/games.js` 注册色觉挑战，桌面宽度展示“请在手机上体验”的提示。
- 新增 smoke test，覆盖文件结构、目录注册、难度递进、色差下限与存储前缀。
