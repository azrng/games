# 2026-03-27 迷宫移动端滚动修复

## 本次目标
- 修复迷宫冒险在移动端点击上下左右按钮时触发页面滚动和出现滚动条的问题。

## 核心改动
- 为 `html`、`body` 增加全屏禁滚和 `overscroll-behavior: none`。
- 为移动端方向键和缩放键补充 `type="button"`，避免默认按钮行为干扰页面。
- 为移动端按钮增加 `pointerdown` / `touchstart` 拦截，主动 `preventDefault()`，并在触发后回到页面顶部。
- 将迷宫容器改为继承根节点尺寸，降低移动端浏览器地址栏伸缩引发滚动条的概率。

- `maze/index.html`
- `TASK.md`

## 风险或遗留项
- 不同移动浏览器对视口高度和触摸事件的实现略有差异，建议在手机浏览器里实际点击上下按钮再确认一轮。
