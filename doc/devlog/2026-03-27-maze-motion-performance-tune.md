# 2026-03-27 迷宫移动流畅度优化

## 本次目标
- 缓解迷宫移动时的卡顿感。
- 降低迷宫边框与角色移动时的抖动。

## 核心改动
- 将迷宫容器和玩家位置更新统一改为基于 `transform` 的位移，减少 `left/top` 带来的重排。
- 增加 `requestAnimationFrame` 调度和视口安全区缓存，避免每一步移动都重复做高频布局测量。
- 对镜头位移做设备像素对齐，并在移动端移除高成本阴影和跟随过渡，减少边缘抖动与拖影。
