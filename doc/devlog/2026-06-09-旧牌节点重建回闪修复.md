# 旧牌节点重建回闪修复

## 本次目标

- 修复动物翻棋翻新牌时，历史已翻开的牌被短暂盖住再翻开的视觉回闪。
- 用 smoke 测试固定旧牌 DOM 不应在后续翻牌时被重建的行为。

## 核心改动

- 将棋盘渲染从每次清空 `board` 后重建 16 张牌，改为只在缺少格子时初始化棋格。
- 后续回合只更新每个棋格的 class 和必要内容，保留已存在 DOM 节点与事件监听。
- 为旧牌节点保留新增回归测试，避免后续改动再次引入整盘重建。
- 升级页面资源 query 到 `20260609-3`，降低浏览器缓存旧脚本的概率。
