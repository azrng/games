# 随机先手与棋子归属修复

## 本次目标

- 去掉开局猜拳弹窗，改为随机决定先手。
- 明确提示当前谁先手、谁回合。
- 修复同一方可能翻出两个相同动物的问题。
- 放慢电脑操作节奏。

## 核心改动

- 开局直接进入对局，随机选择你或电脑先手，并在顶部与底部提示中显示“你先手 / 电脑先手”。
- 初始牌堆改为你和电脑各一套 8 个动物，背面不暴露归属，翻开后显示“你 / 电脑”角标。
- 翻牌不再把当前行动方写成牌归属，只揭示牌本身预设归属。
- 电脑行动延迟从 700ms 调整为 1300ms。
- 删除猜拳弹窗、猜拳 JS 逻辑和相关 CSS。
- 补充 smoke 用例覆盖随机开局、双方各一套动物、翻牌保留预设归属和回合提示。
