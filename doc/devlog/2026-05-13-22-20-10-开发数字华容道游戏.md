# 2026-05-13 开发数字华容道游戏

## 本次目标
- 按 `doc/requirement.md` 中的数字华容道需求，实现移动端 N-Puzzle / 15 Puzzle 风格小游戏并接入游戏平台目录。

## 核心改动
- 新增 `sliding-puzzle/index.html`、`style.css`、`script.js`，实现规格选择、棋盘渲染、点击/滑动移动、撤销、打乱、重置和通关结算。
- 支持 3×3、4×4、5×5 三种规格，使用逆向合法移动打乱，保证局面可解。
- 使用 CSS Grid 渲染棋盘，不使用 Canvas。
- 使用 `sliding-puzzle-{size}-best-steps` 与 `sliding-puzzle-{size}-best-time` 保存每种规格的历史最优。
- 在 `data/games.js` 注册数字华容道，并在 README 当前游戏列表中补充入口。
- 新增 smoke test，覆盖目录注册、移动端入口、规格选项、移动规则、可解性、打乱和历史记录。
