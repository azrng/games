## 迷宫冒险说明

### 当前玩法
- 选择简单、中等、困难三档难度进入游戏。
- 使用键盘方向键或 `WASD` 控制角色移动。
- 移动端提供虚拟方向键和缩放按钮。
- 找到终点处的伙伴后进入下一关。

### 难度设计
- 简单：`13 x 13` 起步，后续每关持续扩张。
- 中等：`19 x 19` 起步，迷宫更大、回溯成本更高。
- 困难：`27 x 27` 起步，视野更紧，路线更长。
- 关卡推进时会同时提升迷宫尺寸，并逐步压缩单格尺寸，所以越往后越难。

### 本次 UI 优化
- 新增顶部 HUD，实时展示关卡、步数、耗时、尺寸和模式。
- 起始难度选择改为卡片式弹层。
- 通关提示改为内嵌结算面板，不再使用原生 `alert`。
- 增加路径访问痕迹，帮助玩家回看已探索区域。

### 运行方式
- 直接在浏览器中打开 `index.html` 即可游玩。
