# 游戏平台

## 项目简介

这是一个以静态页面为主的小游戏合集仓库，当前入口页使用统一游戏清单驱动，支持在首页中切换并加载不同子游戏。

## 技术组成

- HTML
- CSS
- JavaScript

## 当前游戏

- 2048
- 五子棋
- 高级五子棋
- 俄罗斯方块
- 迷宫冒险

## 新增游戏接入规范

新增游戏时，优先遵循以下目录结构：

```text
/<game>/
├── index.html      # 必选，游戏入口页
├── cover.*         # 可选，封面图
├── readme.md       # 可选，游戏说明
└── meta.json       # 可选，扩展元数据
```

接入步骤：

1. 新建游戏目录，并确保入口文件是 `/<game>/index.html`
2. 在 `data/games.js` 中新增一条游戏清单记录
3. 如果游戏需要持久化数据，统一使用 `js/store.js`
4. localStorage 键名统一使用 `app_game_<slug>_<field>` 格式

## 统一清单字段

`data/games.js` 中每个游戏对象当前维护以下字段：

- `slug`
- `title`
- `path`
- `icon`
- `desc`
